# RPM

Running Pace Music coaching service

## Getting Started

To run this project:

```bash
flutter pub get
flutter run -t lib/first.dart
```
## Description

RPM (Running Pace Music)은 러너가 설정한 BPM에 맞춰 음악과 메트로놈을 제공하는 앱입니다. 

## Tech Stack

- **Flutter**: Cross-platform UI toolkit
- **Dart**: 개발 언어
- **Android Studio**: 주요 개발 툴
- **Firebase**
